# Програма практичної роботи №2
Тема: Створення програми на об’єктно-орієнтованій архітектурі.
Мета: Оволодіти новими знаннями про класи та об’єкти на практиці.

Варіант №12
Слово: Слово, Номери сторінок, на яких слово зустрічається (від 1 до 10),
Число сторінок. Створити масив об’єктів. Вивести:
12.1. слова, що зустрічаються більш ніж на N сторінках;
12.2. слова за абеткою;
12.3. для заданого слова номери сторінок, де воно зустрічається.
